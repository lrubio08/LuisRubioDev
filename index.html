¿Quién es Luis?

"Un desarrollador Full Stack apasionado por la tecnología y la programación".

Recientemente completé un bootcamp intensivo de Full Stack donde adquirí habilidades en una variedad de tecnologías, incluyendo Python con Django, HTML, CSS, JavaScript, Bootstrap y bases de datos PostgreSQL.

Durante el bootcamp, trabajé en varios proyectos que me permitieron aplicar y profundizar en mis conocimientos técnicos. Aprendí a construir aplicaciones web desde cero, desde el diseño de la base de datos hasta la creación de interfaces de usuario atractivas, funcionales y responsivas.

Actualmente estoy inmerso en un bootcamp intensivo de Java, donde estoy adquiriendo una sólida base en los fundamentos de la programación orientada a objetos, estructuras de datos y algoritmos . Estoy particularmente entusiasmado con las posibilidades que ofrece Java para desarrollar aplicaciones robustas y escalables. A medida que avanzo en mis estudios, estoy ansioso por aplicar mis conocimientos en proyectos prácticos y contribuir a equipos de desarrollo.

Me encanta el desafío de resolver problemas complejos y la satisfacción de construir algo desde cero. Siempre estoy buscando oportunidades para aprender y crecer como desarrollador. ¡Espero tener la oportunidad de trabajar contigo en el futuro!

Aplicando lo aprendido

TaskSmart | Gestor de Tareas Pendientes

Gestor de Tareas Pendientes, una herramienta diseñada para simplificar la organización y administración eficiente de tus tareas diarias. Con esta aplicación, puedes crear, eliminar y gestionar tus tareas de manera intuitiva, manteniendo un control total sobre tus responsabilidades.

CinePlus utiliza la poderosa API de TMDB para ofrecerte una experiencia cinematográfica completa. Descubre las últimas películas en cartelera, explora trailers, podras conocer el talentoso reparto de cada película y mucho más.
